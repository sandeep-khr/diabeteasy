# diabeteasy



# Getting Started
### Prerequisites
To run Diabeteasy on your local device, you will need to have the following installed:

- Python 3
- Pip

### Installation
Install the required packages
```
pip install -r requirements.txt

```
start the app

```
pyhton app.py

```
